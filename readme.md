# Example: Express with MongoDB

This is an example server running with MongoDB.

## Installation

1. Fork & clone

1. `cp nodemon.sample.json nodemon.json`

1. Create a new Cluster on [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) titled something like `general-purpose` or reuse one you already have.

1. Update the `MONGO_DB_CONNECTION` in your `nodemon.json` file with the full connection string. Make sure you include the password you set up for the database and change the name of the database from `test` to something lke `pies_server_dev`.

1. `npm install`

1. `npm run dev`

Once installation is working, try creating and requesting resources.
